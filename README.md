# Chrome Extension Icon Generator

Generate icons for your Chrome extensions in various required sizes

## Features

- Drag and drop an image or select a file to upload
- Generates icons in 16x16, 32x32, 48x48, and 128x128 sizes
- Preview all generated icons
- Download all icons with a single click
- Copy the manifest.json code snippet for easy integration

## Tech Stack

- [React 18](https://react.dev/) + [Vite](https://vitejs.dev/)
- [Tailwind CSS](https://tailwindcss.com/)
- [lucide-react](https://lucide.dev/)

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server
npm run dev

# Build for production
npm run build
```
